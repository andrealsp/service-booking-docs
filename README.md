# Service Booking Platform – NEXUS

## Overview

NEXUS is a service booking platform designed to explore secure
frontend–backend integration using modern authentication practices.

The current scope focuses on user authentication and session management,
serving as a technical foundation for future booking-related features.

## Goals

- Provide a secure authentication flow
- Demonstrate frontend–backend integration using JWT
- Serve as a portfolio-grade reference for clean architecture practices

## In Scope

- User registration
- User authentication
- Token-based session validation
- Secure API consumption from frontend

## Out of Scope (Current Phase)

- Booking business logic
- Payments
- Notifications
- Role-based authorization beyond basic roles

## Target Audience

- Technical recruiters
- Engineering managers
- Software engineers interested in authentication flows
